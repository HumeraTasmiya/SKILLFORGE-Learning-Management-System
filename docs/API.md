# API Overview

Swagger UI is served at `/api/docs`.

Core modules:

- `/api/auth` registration, login, refresh, logout, password reset, verification
- `/api/courses` catalog, lessons, reviews, enrollment surfaces
- `/api/quizzes` quiz attempts and grading
- `/api/certificates` paid certificate creation, verification, PDF download
- `/api/chatbot` context-aware AI learning assistant
- `/api/analytics` dashboards and engagement metrics
- `/api/payments` Stripe, Razorpay, PayPal certificate checkout flows
- `/api/notifications` real-time and persisted notifications
